## 1.2 Networking components and devices (own words)

- **NIC (Network Interface Card):** Hardware that lets a device join a network and communicate using MAC addressing.
- **Hub:** Simple device that broadcasts incoming data to all ports; no filtering or intelligence.
- **Switch:** Connects devices in a LAN and forwards data using MAC table lookup, reducing unnecessary traffic.
- **Router:** Connects different networks and forwards packets using IP routes.
- **Modem:** Converts digital and analog signals for internet connectivity depending on medium.
- **Access Point (AP):** Allows wireless devices to connect to a wired LAN.
- **Repeater:** Regenerates weak signals to increase transmission distance.
- **Bridge:** Connects LAN segments and filters frames based on MAC address.
- **Gateway:** Protocol translator between networks using different communication rules.
- **Firewall:** Security device/software that permits or blocks traffic according to policy.
- **UTP/CAT6 Cable:** Twisted pair cable used widely for Ethernet LAN connectivity.
- **Patch Panel:** Structured cable management termination point in network racks.
- **Server:** Provides centralized services (files, web, database, authentication) to clients.
- **Client:** End-user machine that requests services from servers over the network.
